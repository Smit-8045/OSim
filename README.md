# OSim – Operating System Simulator

A modern, interactive web application for learning and visualizing core operating system concepts such as CPU Scheduling, Memory Management, and Deadlock Detection. Built with React, Vite, and Tailwind CSS.

## Overview
OSim is designed to help students and enthusiasts understand operating system algorithms through hands-on simulations and visualizations. The platform includes modules for CPU scheduling, memory allocation, deadlock detection, and more.

## Features
- **CPU Scheduling Simulator**: Visualize and compare FCFS, SJF, Round Robin, and Priority algorithms.
- **Memory Management**: Explore memory allocation strategies and paging systems.
- **Deadlock Detection**: Understand and simulate deadlock scenarios and prevention techniques.
- **Interactive Tutorials**: Step-by-step guides for all major OS concepts.
- **Modern UI**: Responsive, clean, and user-friendly interface with Tailwind CSS.

## Getting Started

### Prerequisites
- Node.js (v16 or higher recommended)
- npm (v8 or higher)

### Installation
1. Clone the repository:
   ```sh
   git clone <your-repo-url>
   cd <your-project-directory>
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
4. Open your browser and go to [http://localhost:5173](http://localhost:5173)

## Scripts
- `npm run dev` – Start the development server
- `npm run build` – Build for production
- `npm run preview` – Preview the production build
- `npm run lint` – Lint the codebase
